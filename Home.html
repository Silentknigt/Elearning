“E-Learning on virtual environment ” which is an open source educational and entertainment website. Our aim is to create such an environment which will make learning interesting and fun. In today’s world, it is very difficult to understand everything correctly because the course is so vast that even for small child it becomes too boring to memorize everything. That’s why we are making this website so that we can merge this boring learning with creativity which can make learning entertaining and can help children to gain knowledge.
